# 📊 Amazon Sales Analysis Dashboard

## 📌 Project Overview

The **Amazon Sales Analysis Dashboard** is an end-to-end data analytics project developed to analyze Amazon shopping sales data and generate meaningful business insights.

This project demonstrates the complete workflow of a real-world data analytics pipeline including:

* Data Cleaning using **Python** and **Excel**
* Data Storage using **MySQL**
* Data Visualization using **Power BI**
* Business Problem Identification
* Solution Recommendations

The project helps analyze customer purchasing behavior, revenue trends, product performance, regional sales distribution, and operational challenges.

# 🧹 Data Cleaning Process

The raw dataset contained:

* Missing values
* Duplicate records
* Incorrect state names
* Mixed uppercase/lowercase text
* Special characters
* Invalid city names
* Inconsistent formatting

## ✔ Data Cleaning Using Python

Performed:

* Missing value handling
* Duplicate removal
* Data standardization
* String formatting
* Invalid character removal
* Date formatting
* Data preprocessing

## ✔ Data Cleaning Using Excel

Performed:

* Flash Fill operations
* Find & Replace cleaning
* State and city name correction
* Uppercase/lowercase standardization
* Removal of unwanted symbols and spaces

After cleaning the dataset:

# 📌 Key Insights

* Medium, Large, and XL sizes generated the highest sales
* T-Shirts and Shirts contributed maximum revenue
* Metro cities generated the majority of orders
* Several orders were cancelled, returned, or damaged
* Some product categories showed very low sales performance

---

# ⚠️ Summarized Problems Identified

* High order cancellation rates
* Increased product returns and refunds
* Damaged and lost shipments during delivery
* Uneven sales distribution across states
* Low sales in some product categories
* Inventory imbalance in product sizes
* Heavy dependency on metro cities for sales
* Low sales performance in rural and Tier-2/Tier-3 regions
* Logistics and shipping inefficiencies
* Lack of predictive analysis for future sales forecasting

---

# ✅ Identified Solutions

* Improve inventory management and stock planning
* Enhance product quality and accurate product descriptions
* Provide better size guides to reduce returns
* Improve packaging and delivery tracking systems
* Expand marketing campaigns in low-performing regions
* Optimize inventory based on customer demand trends
* Increase focus on Tier-2 and Tier-3 city markets
* Introduce regional offers and promotional campaigns
* Strengthen logistics and supply chain operations
* Implement predictive analytics and forecasting models
* Use data-driven decision-making for business growth
* Improve customer satisfaction through faster deliveries and reduced cancellations
